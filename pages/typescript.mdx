# TypeScript

TypeScript est un **superset de JavaScript** qui ajoute le typage statique, permettant une meilleure sécurité et maintenabilité du code. Il est particulièrement utile dans les projets de grande envergure ou en équipe.

## Pourquoi utiliser TypeScript ?

1. **Typage statique** : détecte les erreurs de type avant l'exécution.
2. **Outils améliorés** : meilleure autocomplétion, navigation et refactoring dans les éditeurs de code.
3. **Interopérabilité** : compatible avec les bibliothèques JavaScript existantes.
